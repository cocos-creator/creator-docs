# fillColor

The `fillColor` property sets or returns the color used for the fill.

## 实例

```javascript
var ctx = node.getComponent(cc.Graphics);
ctx.fillColor = new cc.Color().fromHEX('#0000ff');
ctx.rect(20,20,250,200);
ctx.stroke();
```
