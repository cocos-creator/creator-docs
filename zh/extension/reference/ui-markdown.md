# ui-markdown 参考

支持显示 mackdown 语法

## 基础用法

```html
<ui-markdown>
    <!-- mackdown 语法的文档 -->
    ## Headers

    # H1
    ## H2
    ### H3
    #### H4
    ##### H5
    ###### H6

    Alternatively, for H1 and H2, an underline-ish style:

    Alt-H1
    ======

    Alt-H2
    ------
</ui-markdown>
```
